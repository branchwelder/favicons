# favicons

Make a favicon.

Live at [https://twigg.gg/favicons/](https://twigg.gg/favicons/).

- 16x16 pixel editor
- mini favicon view
- download favicon
- show favicon in toolbar
- grid overlay
- palette editors: brush, flood, line, rect, shift
- color palette
  - drawn colors update
  - transparency
  - add/delete/edit color in color palette
